# 🩺 Diabetes Prediction using Machine Learning

This project is a beginner-friendly Machine Learning model that predicts whether a person has diabetes based on medical attributes. It uses multiple classification algorithms and compares their performance.

## 📊 Dataset
- Dataset: Pima Indians Diabetes Dataset
- File used: diabetes.csv
- Features include:
  - Pregnancies
  - Glucose
  - Blood Pressure
  - Skin Thickness
  - Insulin
  - BMI
  - Diabetes Pedigree Function
  - Age
- Target Variable:
  - Outcome (0 = No Diabetes, 1 = Diabetes)

## ⚙️ Feature Engineering
- Replaced invalid zero values with mean
- Cleaned dataset for better model performance
- Split data into training and testing sets (80:20)

## 🤖 Models Used
- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier
